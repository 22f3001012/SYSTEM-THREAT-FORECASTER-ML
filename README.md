# SYSTEM-THREAT-FORECASTER-ML PROJECT
ML model to predict malware families using telemetry data

Built classification models to predict malware family infections using telemetry and threat log data.

## Work done
- Data cleaning and preprocessing  
- Feature engineering  
- Model training (classification models)  
- Evaluation using accuracy, precision, ROC-AUC  

## Tools used
Python, Pandas, NumPy, Scikit-learn
